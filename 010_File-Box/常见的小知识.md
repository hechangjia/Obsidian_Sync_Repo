# 半角和全角

> [!note] 半角和全角
> `半角`:正常状态下,字母,数字,符号都只占用一个标准字符的位置
> `全角`:占用两个标准字符的位置.

# 常见术语
`SyntaxError:`语法错误
`Unexpected Error:`意外的缩进.
