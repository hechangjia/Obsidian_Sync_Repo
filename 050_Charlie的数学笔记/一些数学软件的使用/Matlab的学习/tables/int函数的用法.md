#table #matlab #计算机 #编程 

| 函数表达式                    | Matlab代码             | 说明                                      |
| ------------------------ | -------------------- | --------------------------------------- |
| $\int f(x)dx$            | $int(f,'x')$         | 求函数f对x的不定积分                             |
| $\int_{a}^{b}f(x)dx$     | $int(f,'x',a,b)$     | 求函数f对x的定积分,积分区间为 $[a,b]$                |
| $\int_{'m'}^{'n'}f(x)dx$ | $int(f,'x','a','b')$ | 求函数f对x的定积分,积分区间为 $['m','n']$ ,m,n为符号表达式 |
